## Overview

Kotha is a scalable property rental platform designed to streamline listings, bookings, and management for both users and staff. 
The project emphasizes clean architecture, robust role separation, and polished UI/UX for a seamless experience.

## Key Features

- Architected modular dashboards for users and employees/admins with Angular routing, lazy loading, and role-based guards.
- Integrated JWT authentication with token refresh flows to ensure secure, uninterrupted access.
- Enabled dynamic property management with availability toggles, edit/delete actions, and ownership tracking.
- Mapped frontend role logic to backend Django/DRF permissions for unified access control.

## Technologies Used

- **Angular**: For building dynamic, component-based frontend.
- **Django**: As the backend framework for handling APIs, business logic, and role-based authentication.
- **PostgreSQL**: For managing relational data securely and efficiently with strong integrity constraints.
- **Django REST Framework**: For building robust, RESTful APIs
