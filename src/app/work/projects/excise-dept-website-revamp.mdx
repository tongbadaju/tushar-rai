## Overview

Revamped the Excise Department website for the Government of Sikkim.

## Key Features

- Built a scalable full-stack solution using Angular, Django, and PostgreSQL to modernize public service delivery.
- Implemented secure, role-based dashboards for applicants, officers, and administrators.
- Designed a fully responsive UI compliant with government accessibility standards.
- Streamlined multi-step workflows for license applications, document submissions, and approvals.
- Integrated dynamic, API-driven forms to enable real-time validation and efficient data handling.

## Technologies Used

- **Angular**: For building dynamic, component-based frontend.
- **Django**: As the backend framework for handling APIs, business logic, and role-based authentication.
- **PostgreSQL**: For managing relational data securely and efficiently with strong integrity constraints.
- **Django REST Framework**: For building robust, RESTful APIs
